Quais os comandos para configurar usuário e email no git?

Os comandos são:

1. git config --global user.email 2. git config --global user.name

Qual a diferença entre git e github?

Git: É um programa local, instalado no computador do usuário/editor,
utilizado para criar ramificações de "sourcecodes". Isso é importante,
pois assim, recuperar versões anteriores desse código se torna mais simples
e evita que exista o "retrabalho" de escrevê-lo novamente.

GitHub: É um serviço online que nos permite a edição conjunta dos códigos,
bem como a construção e manutenção de um portfólio. Na prática, utiliza-se o git para
construir ramificações que serão salvas tanto no computador quanto no GitHub.
