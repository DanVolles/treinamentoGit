Aqui vamos estudar algumas formas de interagir com a plataforma git.

Quais os comandos para configurar usuário e email no git?

Os comandos são:

1. git config --global user.email 2. git config --global user.name

Qual a diferença entre git e github?

Git: É um programa local, instalado no computador do usuário/editor,
utilizado para criar ramificações de "sourcecodes". Isso é importante,
pois assim, recuperar versões anteriores desse código se torna mais simples
e evita que exista o "retrabalho" de escrevê-lo novamente.

GitHub: É um serviço online que nos permite a edição conjunta (em equipe)
dos códigos, bem como a construção e manutenção de um portfólio. Na prática,
utiliza-se o git para construir ramificações que serão salvas tanto no computador
quanto no GitHub.

O que faz o comando commit?

Esse comando nos permite salvar uma alteração no arquivo. Quando se "commita"
um sourcecode, está-se dizendo que um certo conjunto de alterações serão permanentes
no sourcecode final.

O que faz o comando push?

Esse é o comando utilizado para enviar ("push") para uma máquina remota
como o GitHub. Com esse comando, nós enviamos nosso "commit" para a nuvem.

O que faz o comando pull?

Esse comando nos permite acessar/baixar ("pull") localmente (no computador pessoal)
repositórios remotos como o GitHub. Com esse comando, nós acessamos um sourcecode
disponível no GitHub para editarmos no nosso computador pessoal.
